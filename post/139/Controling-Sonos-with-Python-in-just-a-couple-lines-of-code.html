Controling Sonos with Python in just a couple lines of code

2020-09-29 • By Ivo

Yes, SoCo allows us to control our sonos system with python

A quick thought: Would it be able to control Sonos with Python? I wasn't really sure if this would be something which can be done easily. Turns out it took me 10 minutes to not only figure out that it's possible, but also to implement what I wanted.

I am currently working from home a bit more then I wanted (hence this blog). Part of my job is taking zoom or team calls spontaneously, where I have to stop the music before picking up the call. How cool would it be to be able to both pause and play music on my Sonos system with a simple script? This is how it's done!

Getting started to remotely adjust my Sonos system

Lets first install the SoCo package

pip3 install soco

And lets make a file that will be home of our code

touch script.py

In this file, we will import the soco library and the sys library. The latter is so we can pass arguments in the command line to be handled with the script.

import soco
import sys

First thing is to find all the Sonos groups on your system. For me, I am interested in the first group which is returned, the one closest to my workspace. Therefore, I specify the first element of the list of Sonos groups.

sonos_list = list(soco.discover())
sonos = sonos_list[0]

Now we have a sonos variable that allows us to control many things! I quickly wrote the following:

if sys.argv[1] == 'play':
    sonos.play()
elif sys.argv[1] == 'pause':
    sonos.pause()
elif sys.argv[1] == 'next':
    sonos.next()
elif sys.argv[1] == 'previous':
    sonos.previous()
elif sys.argv[1] == 'vol_up':
    sonos.volume += 10
elif sys.argv[1] == 'vol_down':
    sonos.volume -= 10

Get it? Running:

python3 script.py play

Will play the current track

python3 script.py pause

Will pause the current track

python3 script.py next

Will play the next track

python3 script.py previous

Will play the previous track (for all the guilty pleasures which you want to repeat)

python3 script.py vol_up

Will increase 1/10th of the volume

python3 script.py vol_down

Will decrease 1/10th of the volume

I did not expect it to be so simple! What I did, is made a workflow with Alfred. What this is exactly, and how you can do it as well can be found here

Full code:

import soco
import sys

sonos_list = list(soco.discover())
sonos = sonos_list[0]

if sys.argv[1] == 'play':
    sonos.play()
elif sys.argv[1] == 'pause':
    sonos.pause()
elif sys.argv[1] == 'next':
    sonos.next()
elif sys.argv[1] == 'previous':
    sonos.previous()
elif sys.argv[1] == 'vol_up':
    sonos.volume += 10
elif sys.argv[1] == 'vol_down':
    sonos.volume -= 10

Have a look at Soco's Guithub page if you'd like to learn more about the functionalities.
