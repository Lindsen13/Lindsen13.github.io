Datetime, Date, UNIX, Timezones and more: A quick overview of the most seeked questions regarding dates and times!

This post will be a bit different from normal posts: With each project, I am always using google to find out how to solve my datetime-issue. I just keep on forgetting how to work with python's datetime library. Therefore, the following questions and answers are the most relevant python-datetime questions according to Stackoverflow.

How to convert "Jun 1 2005 1:33PM" to a datetime object? (read from this post)

from datetime import datetime

datetime_object = datetime.strptime('Jun 1 2005  1:33PM', '%b %d %Y %I:%M%p')
print(datetime_object)

gives

2005-06-01 13:33:00

How to get the current datetime in python? (read from this post)

import datetime

datetime_object = datetime.datetime.now()
print(datetime_object)

gives

2020-07-27 19:55:08.238399

How to get the current time in python? (read from this post)

import datetime

datetime_object = datetime.datetime.now().time()
print(datetime_object)

gives

19:55:42.884658

How to convert a date object into a datetime object? (read from this post)

import datetime

datetime_object = datetime.datetime.combine(datetime.date.today(), datetime.datetime.min.time())
print(datetime_object)

gives

2020-07-27 00:00:00

How to parse an ISO 8601-formatted date? (read from this and this post.)

import datetime

datetime_object = datetime.datetime.fromisoformat('2017-01-01T12:30:59.000000')
print(datetime_object)

gives

2017-01-01 12:30:59

How to subtract a day from a date? (retrieved from this post)

import datetime

days_to_subtract = 10
datetime_object = datetime.datetime.today() - datetime.timedelta(days=days_to_subtract)
print(datetime_object)

gives

2020-07-17 20:10:32.507353

In the same fashion, we can also subtract a specified amount of seconds:

import datetime

seconds_to_subtract = 3600
datetime_object = datetime.datetime.today() - datetime.timedelta(seconds=seconds_to_subtract)
print(datetime_object)

gives

2020-07-27 19:13:39.340982

How to print a datetime object without microseconds? (retrieved from this post)

import datetime

datetime_object = datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")
print(datetime_object)

gives

2020-07-27 20:15:05

We can also use this to change the format of the datetime object:

import datetime

datetime_object = datetime.datetime.now().strftime("%Y/%m/%d hour: %H minute: %M second: %S")
print(datetime_object)

gives

2020/07/27 hour: 20 minute: 16 second: 29

How to get the weekday from a datetime object? (retrieved from this post)

import datetime

datetime_object = datetime.datetime.today().weekday()
print(datetime_object)

gives

0

Note that monday is 0 and sunday is 6.

How to get the weekday in english from a datetime object? (retrieved from this post)

import datetime

datetime_object = datetime.datetime.now().strftime('%A')
print(datetime_object)

gives

'Monday'

How to convert a UNIX timestamp into a datetime object? (retrieved from this post)

import datetime

datetime_object = datetime.datetime.fromtimestamp(1172969203.1)
print(datetime_object)

gives

2007-03-04 01:46:43.100000

This one is specifically useful when working with APIs, often you get a UNIX timestamp returned. When reading a bit more at the original thread, it came appeared that Python will return the timestamp converted to the timezone you are in. To make sure you always convert it to the same timestamp (UTC):

import datetime

datetime_object = datetime.datetime.utcfromtimestamp(1172969203.1)
print(datetime_object)

gives

2007-03-04 01:46:43.100000

Thanks!
