Running Python script with Alfred, the productivity booster!

2020-08-25 • By Ivo

Alfred used to run any Python script and parse the output to you!

One of my favourite programs for my mac is Alfred. It is productivity-booster that can be executed with custom defined hotkeys. Part of Alfred is the possibility to execute workflows, which can be made by anyone. One of the nodes is the run script action. This allows you to run Bash, Ruby, Pearl, and Python!

I will show how you can execute a python script with Alfred. First, we need a Python script. I made this short script that scrapes the latest post of this website:

import requests
from bs4 import BeautifulSoup

output = requests.get('https://python-everything.com/')
soup = BeautifulSoup(output.text, 'html.parser')

newest_post = soup.find(class_='feature-1 text-center').text.strip()

If you want to use this script, make sure you have BeautifulSoup installed:

pip install BeautifulSoup4

Next, we need to open Alfred's preferences and go to Workflows. Here we create a new workflow, which will be very simple. By using a single command, we can execute a Python script and parse an output to show on the screen.

The first node, defines which word will trigger our script. In this case, it is Python.

Next, we need to add the script we made before into the second node. We also need to add two extra lines:

import sys
#original code here
sys.stdout.write(newest_post)

The sys.stdout.write(newest_post) argument will parse the newest_post variable to the programm which calls the Python script, in this case Alfred.

This variable is used in the last node of the workflow, which produces an message for us:

Now if we trigger Alfred and type Python:

We get the name of the latest post on Python Everything!

You can download the workflow here to add it to Alfred yourself.
