Creating Classes to handle your Python code!

2020-08-24 • By Ivo

How a Python Class helps us to scrape data Youtubers with Requests and BeautifulSoup

After diving into the Alteryx API wrapper package I found, I have been looking at the structure on how the API calls are executed.

One of the main difference I found with the Python code I write most of the time, is the use of Classes. Most of the time, I write some Python code, wrap it into one or two functions, put a if __name__ == "__main__": line at the end of the file so we can import the functions, and thats it.

For this project, I'd like to explore how I could use the Python Classes to make my computer do what I want. This time, it is going to be scraping Youtube channels for their stats!

First, lets define a class for our scraper:

class Youtube:
    def __init__(self, url: str):
        self.url = url
        self.headers = {"Accept-Language": "en-US,en;q=0.5"}

We are creating a class, where we define the URL we want to scrape, and we store a header variable for our requests we are about to make. Furthermore, I will add a quick logic to see if people send in the proper url and a short string to explain what my Class will do.

class Youtube:
    """
    Class to scrape Youtube channel
    """
    def __init__(self, url: str):
        self.url = url
        if not re.match(r'.*w*/c/.*/videos',self.url):

        self.headers = {"Accept-Language": "en-US,en;q=0.5"}

Next up is to download the unstructured data from Youtube with the Request library and the BeautifulSoup4 library. 

import requests
from bs4 import BeautifulSoup
import re
import json

def scrape(self):
    output = requests.get(self.url, headers=self.headers)
    soup = BeautifulSoup(output.text, 'html.parser')
    output = [str(element) for element in soup.find_all('script') if 'window["ytInitialData"]' in str(element)][0]
    json_output = json.loads(re.search(r'ytInitialData"]s{1}.{1}s{1}(.*);', output).group(1))

We are sending the output of the get request to the BeautifulSoup library, which we then parse with a list comprehension. This list comprehension returns the html-script tag that contains the info which we need. With some regex magic we remove everything which we cannot convert to a JSON format. What we are left over is json_output:

Each method checks if the variable yt_content is populated with data. If not, it will fetch the data with the defined scrape() function. otherwise, it parses the content to retrieve the method which is needed. To make it nice and pythonic, I used list comprehensions to do so.

Putting all the elements together gives us:

import requests
from bs4 import BeautifulSoup
import re
import json

class Youtube:
    """
    Class to scrape Youtube channel
    """
    def __init__(self, url: str):
        self.url = url
        if not re.match(r'.*w*/c/.*/videos',self.url):
            raise Exception(f"Wrong url {self.url} specified. It needs to have the following format: 'https://www.youtube.com/c/accountname/videos'")
        self.headers = {"Accept-Language": "en-US,en;q=0.5"}
        self.yt_content = []

    # Returning a list of the last 30 video titles
    def titles(self):
        if self.yt_content == []:
            self.scrape()
        return [video.get('title') for video in self.yt_content]

    # Returning a list of the last 30 video ids
    def id(self):
        if self.yt_content == []:
            self.scrape()
        return [video.get('id') for video in self.yt_content]

    # Returning a list of the last 30 video long titles, consistent of title, date, viewcount.
    def long_title(self):
        if self.yt_content == []:
            self.scrape()
        return [video.get('long_title') for video in self.yt_content]

    # Returning a list of the last 30 video published-at dates
    def published_at(self):
        if self.yt_content == []:
            self.scrape()
        return [video.get('published') for video in self.yt_content]

    # Returning a list of the last 30 video viewcounts
    def viewcount(self):
        if self.yt_content == []:
            self.scrape()
        return [video.get('viewcount') for video in self.yt_content]

    # Returning a list of the last 30 video urls
    def url(self):
        if self.yt_content == []:
            self.scrape()
        return [video.get('url') for video in self.yt_content]

    # Returning a list of the last 30 video durations
    def duration(self):
        if self.yt_content == []:
            self.scrape()
        return [video.get('duration') for video in self.yt_content]

    # Returning a list of the last 30 videos as a dictionary
    def complete_overview():
        if self.yt_content == []:
            self.scrape()
        return self.yt_content

    # Function that scrapes the data
    def scrape(self):
        yt_info = []
        output = requests.get(self.url, headers=self.headers)
        soup = BeautifulSoup(output.text, 'html.parser')
        output = [str(element) for element in soup.find_all('script') if 'window["ytInitialData"]' in str(element)][0]
        json_output = json.loads(re.search(r'ytInitialData"]s{1}.{1}s{1}(.*);', output).group(1))
        videos = json_output.get('contents').\n        get('twoColumnBrowseResultsRenderer').\n        get('tabs')[1].\n        get('tabRenderer').\n        get('content').\n        get('sectionListRenderer').\n        get('contents')[0].\n        get('itemSectionRenderer').\n        get('contents')[0].\n        get('gridRenderer').\n        get('items')
        for video in videos:
            yt_info.append({
                'id':video.get('gridVideoRenderer').\n                    get('videoId'),
                'title': video.get('gridVideoRenderer').\n                    get('title').\n                    get('runs')[0].\n                    get('text'),
                'long_title': video.get('gridVideoRenderer').\n                    get('title').\n                    get('accessibility').\n                    get('accessibilityData').\n                    get('label'),
                'published': video.get('gridVideoRenderer').\n                    get('publishedTimeText').\n                    get('simpleText'),
                'viewcount': video.get('gridVideoRenderer').\n                    get('viewCountText').\n                    get('simpleText'),
                'url': video.get('gridVideoRenderer').\n                    get('navigationEndpoint').\n                    get('commandMetadata').\n                    get('webCommandMetadata').\n                    get('url'),
                'duration': video.get('gridVideoRenderer').\n                    get('thumbnailOverlays')[0].\n                    get('thumbnailOverlayTimeStatusRenderer').\n                    get('text').\n                    get('accessibility').\n                    get('accessibilityData').\n                    get('label')
                })
        self.yt_content = yt_info

Let's assume we are interested in finding out what Casey Neistat has been publishing lately. We will paste the URL in a url variable, assign the Youtube Class to a variable, and call the titles() method to retrieve all the titles from Casey Neistat's latest videos.

One way to do this, is by using Selenium to scrape the website, instead of Python's Requests library. With Selenium, you are able to execute browser-like behavior, like scrolling down. Doing this would enable us to scroll down to the bottom to retrieve information of all the videos a certain user has posted.
