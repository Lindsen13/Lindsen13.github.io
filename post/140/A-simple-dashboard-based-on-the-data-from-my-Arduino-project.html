A simple dashboard based on the data from my Arduino project

2020-11-05 • By Ivo

Using Tableau and my previous IoT devices build with Ardunio and Google Cloud Functions to create amazing dashboard

Remember, in one of my last posts I made a simple device that tracks the temperature in the room. I have right now 3 of them running, all writing data to the same SQL database with the help of Google Cloud functions.

In this post, I'd quickly would like to show what I did with this data. Although I did not have to use any Python code, it's still something which is pretty cool.

A simple dashboard with Tableau Public

I made a simple dashboard with this data, and uploaded it to Tableau public. Now I can always see what the temperature is in my house, for different areas.

You can see that I have placed an arduino in my living room, in my workshop and in my 3d printing enclosure. You can click on the different signs on the right to switch between the three locations. So much fun to see my personal behavior back in this data!

As always, thanks for reading!
