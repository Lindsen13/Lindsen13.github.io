Converting Markdown text to HTML

2020-07-15 • By Ivo

How Markdown can be converted with Python to HTML without any effort!

While creating this application with the Flask framework (I am sure future posts will leverage the knowledge I gained in the process of making this site), I came across the markdown package. As a test on how good this package works, I will describe how I used it for creating a blog-post-generator for this app!

Lets create a folder, create a virtual environment and install the package:

# create a directory called 'markdown-sample'
mkdir markdown-sample
# Navigate into the directory
cd markdown-sample
# Create a virtual environment
python3 -m venv env
# Activate the virtual environment
source env/bin/activate
# Install the 'Markdown' package.
pip install Markdown
# Create a python file
touch mardkown_example.py

Now we have our python file and environment in place, we can edit the file and start playing around!

from markdown import markdown
text = "#This is a test"
output = markdown(text)
print(output)

This should return:

<h1>This is a test</h1>

or after rendering:

This is a test

Now lets include some python code ....

from markdown import markdown
text = """
    print('hello world!')
"""
output = markdown(text)
print(output)

which returns (after putting some indents in place):

<pre>
    <code>
        print('hello world!')
    </code>
</pre>

or:

print('hello world!')

and it works! Great! There is only one issue. You might be able to fit it when we run the following:

from markdown import markdown
text = """
    for i in range(100):
        if i < 50:
            print(f'the value of i is still small: {str(i)}')
        else:
            print(f'the value of i is big: {str(i)}')
"""
output = markdown(text)

Which returns:

<pre>
    <code>
        for i in range(100):
            if i < 50:
                print(f'the value of i is still small: {str(i)}')
            else:
                print(f'the value of i is big: {str(i)}')
    </code>
</pre>

or: 

for i in range(100):
    if i < 50:
        print(f'the value of i is still small: {str(i)}')
    else:
        print(f'the value of i is big: {str(i)}')

Can you see it? There is no formatting for the code! Now imagine reading a bunch of python code without some nice colour formatting to see the difference between a method and a function... Luckily we can use an extension to overcome this issue. We will be using the codehilite extension, which allows us to generate classes for the markdown we are writing. Let's install it an run it with the extension

pip install Pygments

and adding a some info to show we are trying to print python code:

from markdown import markdown
text = """
    ::python
    for i in range(100):
        if i < 50:
            print(f'the value of i is still small: {str(i)}')
        else:
            print(f'the value of i is big: {str(i)}')
"""
output = markdown(text, extensions=['codehilite'])
