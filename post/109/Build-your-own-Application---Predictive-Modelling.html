Adding to our environment

Before we do so, I'd like to point out that my prediction model is poorly chosen, and is not fine-tuned at all. This post and the application is merely a showcase to show what's possible. We are creating a forecast model, which we will load in our front-end application to create a forecast based on the user's input. Lets take care of our create_model() function first.

df['price'] = pandas.to_numeric(df['price'], errors='coerce')

Now we have a Pandas data frame with all the required info. We will use a linear regression model to make our predictions. For this, we cannot use any categorical values. Luckily, Pandas has a get_dummies() function that converts all categorical (string) values into dummies. This means that rather than having a column with the brands, we will have a column for each possible brand, and a 1 or a 0 if it's the brand of the car.

df = pandas.get_dummies(df)

Before we get started with our model, we have to make sure there are no empty values in our data frame:

df = df.dropna()

Great! Now we will make a regression model. For this, we need an array of the value we want to predict (price, which we will cal y) and an array of variables

y = np.array(df['price'])
df = df.drop(['price'], axis=1)
x = np.array(df)

With this, we can create a regression model

model = LinearRegression().fit(x, y)

At this point, you most likely want to check how good your model is working. Since this series is just a showcase on how you could make a web app with predictive capabilities, I will not go over this.

Great! All we have to do is create a list of all variables in our model (all column names of the df data frame) and store the model so we can call it later.

Putting this together gives:

def create_model():
    db_file = 'instance/database.db'
    conn = create_connection(db_file)
    cur = conn.cursor()
    rows = cur.execute("SELECT * FROM cars;")
    rows = cur.fetchall()
    df = pandas.DataFrame(rows, columns =['id', 'date', 'name','price','brand','type','year','km','fuell','gearbox'])
    df = df[['price','brand','year','km','fuell','gearbox']]
    df['year'] = pandas.to_numeric(df['year'].str[:4])
    df['km'] = pandas.to_numeric(df['km'], errors='coerce')
    df['price'] = pandas.to_numeric(df['price'], errors='coerce')
    df = pandas.get_dummies(df)
    df = df.dropna()

    y = np.array(df['price'])
    df = df.drop(['price'], axis=1)
    x = np.array(df)

    model = LinearRegression().fit(x, y)
    names = [x for x in df.columns]
    pickle.dump(model, open('instance/finalized_model.sav', 'wb'))
    return names

Furthermore, I want to be able to run the util.py file to create the model, and to return a prediction for an entry. Therefore, I add the following to the end of the util.py file

if name == 'main': data = { 'year':2011, 'km':1000, 'brand':'audi', 'fuell':'Benzine', 'gearbox':'Automatisch' } names = create_model() model = pickle.load(open('instance/finalized_model.sav', 'rb')) output = create_prediction(data, model, names) print(output)

Every time we run the util.py file, the model is run, the model is stored and a prediction is made for the data dictionary.

Note: I am aware that my model has some strange inputs: it's Dutch. I have scraped a Dutch website, hence I have to put in Dutch words. The example above inserts an Audi from 2011 with 1000 km on the clock, running on benzin and has a manual gearbox.

import pandas

def convert_date(output):
    monthyear = output.find(class_='css-tr15v5').find('b').text
    year = monthyear.split(' ')[-1]
    month = monthyear.split(' ')[0]
    if month =='Jan':
        month = '01'
    elif month =='Feb':
        month = '02'
    elif month =='Mrt':
        month = '03'
    elif month =='Apr':
        month = '04'
    elif month =='Mei':
        month = '05'
    elif month =='Jun':
        month = '06'
    elif month =='Jul':
        month = '07'
    elif month =='Aug':
        month = '08'
    elif month =='Sep':
        month = '09'
    elif month =='Okt':
        month = '10'
    elif month =='Nov':
        month = '11'
    elif month =='Dec':
        month = '12'
    else:
        month = '12'
    return f'{year}-{month}-01'
